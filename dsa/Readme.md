# DSA Interview Prep – Frequently Asked Problems

This list contains ONLY high-frequency Data Structures & Algorithms problems
commonly asked in technical interviews (FAANG + product companies).

---

## 1. Arrays & Strings (Highest Priority)

1. [Two Sum](1.1-two-sum.go)  
2. Maximum Subarray (Kadane’s Algorithm)  
3. Longest Substring Without Repeating Characters  
4. Best Time to Buy and Sell Stock  
5. Product of Array Except Self  

Patterns:
- Hashing
- Sliding Window
- Greedy
- Prefix logic

---

## 2. Linked Lists

1. Reverse Linked List  
2. Linked List Cycle (Fast & Slow Pointer)  
3. Merge Two Sorted Lists  
4. Remove Nth Node From End  

Patterns:
- Pointer manipulation
- Two-pointer technique

---

## 3. Stack

1. Valid Parentheses  
2. Min Stack  
3. Daily Temperatures  

Patterns:
- Stack
- Monotonic Stack

---

## 4. Hashing

1. Contains Duplicate  
2. Group Anagrams  
3. Longest Consecutive Sequence  

Patterns:
- Hash Map
- Hash Set
- Frequency counting

---

## 5. Trees (Binary Trees Only)

1. Maximum Depth of Binary Tree  
2. Level Order Traversal  
3. Diameter of Binary Tree  
4. Lowest Common Ancestor  

Patterns:
- DFS (recursion)
- BFS (queue)

---

## 6. Binary Search

1. Binary Search  
2. Search in Rotated Sorted Array  
3. Find Minimum in Rotated Sorted Array  

Patterns:
- Sorted search space
- Binary search on answer

---

## 7. Graphs (Minimal & High-ROI)

1. Number of Islands  
2. Course Schedule  

Patterns:
- DFS / BFS
- Cycle detection
- Topological thinking

---

## 8. Dynamic Programming (Minimal Set)

1. Climbing Stairs  
2. House Robber  
3. Coin Change  

Patterns:
- 1D DP
- State transition
- Optimization

---

## Notes

- Always explain:
  - Approach
  - Time Complexity
  - Space Complexity
- Focus on recognizing patterns quickly
- Code cleanly with edge cases
